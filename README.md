# e-plantShopping

**Project Name:** e-plantShopping (Paradise Nursery)

**Description:**
A React-based e-commerce application for Paradise Nursery. This project allows users to browse houseplants by category, add them to a shopping cart, manage quantities, and view their cart. It utilizes Redux for state management and React Router for navigation.

## Features
- Browse plants by category
- Add to cart functionality
- Dynamic cart management (increment, decrement, delete)
- Responsive design
